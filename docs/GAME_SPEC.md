# GAME_SPEC

このプロジェクトは、リアルタイム進行の 1vs1 カード戦闘を中心にしたローグライトです。

## コアループ

1. スターターロードアウトを選ぶ
2. 敵とリアルタイム戦闘を行う
3. 勝利時に報酬カードを選ぶ
4. 数戦後にボスへ到達する
5. 結果を集計して Hub に戻る

## 戦闘の要点

- カードは `cast_time` 経過後に発動する
- 発動後は `recast_time` の間再使用できない
- 同時に展開できるカード総量は `active_slot_max = 3`
- 発動順は `scheduled_time` と `priority_modifier` で決定する
- `Space` 長押しで Slow Mode が有効になる

## MVP の主な対象

- 15 枚のカード
- 5 体の敵
- プレイヤー手動操作
- 敵 AI による自動行動
- 戦闘ログとタイムライン表示
